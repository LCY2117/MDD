# 心语社区 - 抑郁症患者支持平台

一个温和、安全、充满陪伴感的抑郁症患者及家属支持社区。

## ✨ 特点

- 🎨 **温和设计** - 低饱和蓝绿色系，柔和圆角，微妙阴影
- 🔒 **隐私保护** - 支持匿名发布，完善的隐私控制
- 🆘 **危机支持** - 紧急求助热线，专业资源推荐
- 💬 **社区互动** - 安全的表达空间，温暖的陪伴氛围
- 🤖 **AI智慧问答** - 专为抑郁症患者设计的智能助手
- 📈 **情绪追踪** - 可视化情绪记录和趋势分析
- 👨‍👩‍👧 **家属支持** - 专门的家属模式，提供沟通和照护指南
- 📱 **移动优先** - 适配微信小程序，响应式设计

## 🎯 用户体验亮点

### 1. 共情的文案
- "你好呀" 而不是 "欢迎"
- "感到特别困难吗？" 而不是 "需要帮助吗？"
- "慢慢来，一切都会好起来" 而不是 "加油"

### 2. 温和的提醒
- 敏感词检测不阻止发布，而是温和建议
- 提供求助资源，而不是警告

### 3. 隐私优先
- 匿名开关显眼易用
- 发布前明确提示隐私状态
- 随时可以删除内容

### 4. 危机干预
- 首页显示求助入口
- 紧急热线可直接拨打
- 多层次的支持资源

## 🌈 设计考量

### 为什么选择低饱和蓝绿色？
- 蓝绿色在色彩心理学中代表平静、希望、治愈
- 低饱和度避免视觉刺激，更温和
- 不同于医疗的冷白，也不同于社交的鲜艳

### 为什么强调"陪伴感"？
- 抑郁症患者常感到孤独
- "陪伴"比"帮助"更平等、更温暖
- 社区的核心价值是相互支持

### 为什么突出隐私？
- 精神健康话题敏感
- 匿名降低分享门槛
- 隐私保护建立信任

## 📝 文案规范

### DO ✅
- 使用第一人称和第二人称（"我们"、"你"）
- 肯定句比否定句好
- 具体比抽象好
- 提供选择而非命令

### DON'T ❌
- 避免"应该"、"必须"
- 避免评判性语言
- 避免过度励志
- 避免medical claim
